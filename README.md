# ToDo List App with Login (Android - Kotlin)

## Description

This project is an Android application developed using Kotlin. It allows users to log in and manage their daily tasks through a simple and easy-to-use interface.

## Features

* Login screen for user input
* Add new tasks
* Delete existing tasks
* Display list of tasks

## Tech Stack

* Kotlin
* Android Studio
* XML for UI design
